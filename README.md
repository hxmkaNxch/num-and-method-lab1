# num-and-method-lab1

Учебный project / лабораторная работа по численным методам и анализу данных в Jupyter Notebook.  
Проект показывает базовые принципы работы с табличными данными, исследовательского анализа, подготовки признаков и построения регрессионной модели для предсказания целевой переменной на основе числовых признаков.

## About Project

**num-and-method-lab1** - это учебный проект, в котором решается задача регрессии на табличном датасете.  
В работе используются train/test выборки, проводится первичный анализ данных, а затем строится модель машинного обучения для предсказания целевого признака. В ноутбуке используются инструменты визуализации, масштабирования и генерации полиномиальных признаков, что делает проект хорошей практикой в обработке данных и базовом моделировании.

Проект был сделан как практика в:
- анализе табличных данных
- работе в Jupyter Notebook
- визуализации данных
- подготовке признаков
- построении регрессионной модели
- использовании инструментов scikit-learn

## Features

- **Dataset Loading** - чтение train и test выборок
- **Exploratory Data Analysis** - первичный анализ структуры и статистик датасета
- **Data Visualization** - использование графиков для изучения данных
- **Feature Preparation** - подготовка признаков перед обучением модели
- **Scaling** - применение масштабирования признаков
- **Polynomial Features** - генерация полиномиальных признаков
- **Regression Modeling** - обучение модели линейной регрессии
- **Train/Test Split** - разбиение данных для обучения и проверки
- **Submission Preparation** - работа с примером итогового файла предсказаний

## Workflow

1. Загружаются train и test данные
2. Проводится просмотр датасета и базовый анализ признаков
3. Выполняется подготовка данных для обучения
4. При необходимости применяются масштабирование и полиномиальные признаки
5. Обучается регрессионная модель
6. Готовятся предсказания для тестовой выборки
7. Формируется итоговый файл результата
